# Conch 海螺

`Conch 海螺` 是「Project Senjuko」子项目之一，作为事件发布者的基础微服务，提供基于 `QQ for Android*` 的通讯封装和实现，基于
[`NATS`](https://nats.io/) 的消息队列和基于 [`GraphQL`](https://graphql.org/) 的 API 查询语言，
致力为微服务及普通开发者打造优良的 QQBot 开发体验。

### WIP 警告

此项目还在早期快速开发中，不代表所述功能已经完成。
